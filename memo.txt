vs code에서 파일을 생성시에는 반드시 파일명 뒤에 .확장자 를 포함해서 작성해야 한다.
Alt + Z 자동 줄바꿈 단축키

-------리눅스 명령어 기초연습-----------

터미널 Ctrl + J -> Git Bash 여는 단축키
1. ~ 최상위 디렉터리 위치
2. pwd 현재 경로 디렉터리(폴더) 표시
3. ls 현재 디렉터리 안에있는 폴더와 파일을 표시
4. mkdir 생성디렉터리명
    현재 경로 위치에 디렉터리 생성
5. rm -r 삭제디렉터리명
    현재 경로 위치에 디렉터리 제거 (내용까지 모두)
6. cd 디렉터리명
    하위 디렉터리로 이동 
7. cd ..
    상위 디렉터리는 이름을 인식할 수 없으므로 한 단계위(..)라는 뜻으로 ..을 사용해서 나간다.
8. vim 생성파일면.확장자
    원하는 파일을 확장자 포함해서 ex)test.txt 생성 후
    i 눌러서 끼워넣기 버전으로 변경하고
    원하는 내용 입력 후 ESC 키 누르고 :wq 저장하고 종류
9. cat 확인파일명.확장자
    내용 확인하고 싶은 파일명 확장자 포함 ex)test.txt 작성 후
    확인하면 내용을 미리 볼 수 있다.

* TIP 
터미널 입력 중 갑자기 입력이 안되거나 입력화면이 보이지 않는 경우
Ctrl + C 또는 Q를 누르면 정상으로 돌아간다.

10. git init
    내가 올리려는 저장소의 위치를 알리는 명령어
11. ls -a 숨긴 파일까지 모두 표시해서 보여라 명령어
12. git status 상태확인
13. git log 누가 몇시 몇분 몇초에 commit을 했는지 알려주는 명령어
